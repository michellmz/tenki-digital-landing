# Tenki Digital Landing
Website of Tenki Digital
This project presents a comprehensive rebranding and modern frontend proposal for Tenki Digital, a company merging technology and sustainable energy.

The goal was to build a modern, high-performance landing page with a dark/vibrant red aesthetic reflecting the brand's dual vision.

Key Features:

Implemented scroll-reveal animations (Intersection Observer).

Ensured 100% responsive design.

Integrated perfectly centered SVG social icons.

The final product offers a dynamic and professional digital presence.
